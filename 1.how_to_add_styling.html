Several ways how to add your CSS

  • Inline - Styles are applied directly to an element in HTML using <style > attribute
  • Embedded - a stylesheet are added within the head of HTML using <style > element
  • External - stylesheet are stored in a separate .css file using <link> element . Recommended !
