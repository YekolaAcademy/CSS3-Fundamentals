Universal snippet

Bellow code universally apply the border-box model to all elements and their pseudo-elements 

        html {
        box-sizing: border-box;
        }
        
        *, *: before, *:after {
        box-sizing: inherit;
        }

Pseudo-classes 

         a:link, a:visited {
        color: deepskyblue;
        text-decoration: none;
        }
         
         a:link, a:active {
        color: blue;
        text-decoration: dashed underline;
        }

Note the order: :link,:visited, :hover, and :active.

Attribute Selectors snippet 

        input[type=text] {
        margin-bottom: 15px;
        margin-bottom: deepskyblue;
        color: white;
        }

Selecting elements part two

  • The Universal Selector(*) - To select all elements throughout an HTML
  • Attribute Selectors [] - To select element based on the present of an attribute or value
  • Pseudo-classes(:) - To select element when it is in a particular state
