Pseudo-element In Action

  • Descendants e.g ul li {color: blue } - Select all the element that are descendants
  • Direct children - Select all the element that are descendants. Not any li elements that are nested. Elements matched by the second selector must be the immediate children of the elements matched by the first selector
  • Next sibling - select element that is a next sibling of li.selected + li. li.selected ~ li “all” of the li element. Next sibling selector (+) is placed between two CSS selectors. It matches only those elements matched by the second selector that are the next sibling element of the first selector.
